### Project Requirements

Create a web page using **HTML** and **CSS** with the following features:

- A heading that describes the purpose of the page
- A list of questions, each followed by its corresponding answer
- The links should be clickable and navigatable with the **Tab** key
- The layout should be clean and visually appealing, with appropriate spacing and typography

This project is for [devchallenges](https://devchallenges.io/), specifically for the [Simple Frequently Asked Questions (FAQ) challenge](https://devchallenges.io/challenge/simple-faq-challenge).
